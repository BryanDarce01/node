# Hacer queries en mongoDB

## 1. `.find()` — lo básico

``` javascript

db.usuarios.find()

```

- Trae todos los documentos de la colección

## 2. Con filtro

``` javascript

db.usuarios.find({ nombre: "Bryan" })

```

- Trae solo los que cumplan esa condición

## 3. Operadores ($gt, $lt, etc.)

Estos son como los operadores en SQL (`>`, `<`, etc.)

### 3.1 `$gt` (greater than = mayor que)

``` javascript

db.usuarios.find({ edad: { $gt: 18 } })

```

- Usuarios con edad mayor a 18

### 3.2 `$lt` (less than = menor que)

``` javascript

db.usuarios.find({ edad: { $lt: 30 } })

```

- Menores de 30

## 4. Combinados

``` javascript

db.usuarios.find({
  edad: { $gt: 18, $lt: 30 }
})

```

- Entre 18 y 30

## 5. `$or`

Se le debe de pasar un `array` al operador `$or` por eso se usan los `[ ]` y adentro van los objetos

``` javascript

db.usuarios.find({
  $or: [
    { nombre: "Bryan" },
    { edad: { $lt: 20 } }
  ]
})

```

Trae:

- los que se llaman Bryan
O los menores de 20

## 6. `{ name: 1 }` → PROYECCIÓN

Esto es importante.

- Sirve para decir qué campos queremos que devuelva.

Ejemplo

``` javascript

db.usuarios.find(
  { edad: { $gt: 18 } },
  { nombre: 1 }
)

```

## 7. Ocultar `_id`

``` javascript

db.usuarios.find(
  {},
  { nombre: 1, _id: 0 }
)

```

# Con SQL

``` sql

SELECT nombre FROM usuarios WHERE edad > 18;

```

# Con mongoDB

``` javascript

db.usuarios.find(
  { edad: { $gt: 18 } },
  { nombre: 1 }
)

```

## Ejemplo real

``` javascript

db.usuarios.find(
  {
    $or: [
      { edad: { $gt: 25 } },
      { nombre: "Ana" }
    ]
  },
  {
    nombre: 1,
    edad: 1,
    _id: 0
  }
)

```

Significa:

- Usuarios con edad > 25 O nombre Ana

- Solo devuelve nombre y edad

# Resumen

- `.find()` → buscar datos

- `{}` → condiciones

- `$gt`, `$lt` → comparaciones

- `$or` → múltiples condiciones

- `{ campo: 1 }` → qué mostrar
