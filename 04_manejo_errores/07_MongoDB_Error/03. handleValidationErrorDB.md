# `handleValidationErrorDB` (error de validación)

``` javascript

const handleValidationErrorDB = err => {
  const errors = Object.values(err.errors).map(el => el.message);

  const message = `Invalid input data. ${errors.join('. ')}`;
  return new AppError(message, 400);
};

```

## ¿Qué error está manejando?

Errores de validación de Mongoose, por ejemplo:

``` javascript

name: {
  type: String,
  required: [true, 'Name is required']
}

```

### Ejemplo

Enviamos:

``` json

{
  "email": "no-es-email"
}

```

### 🔴 Mongoose genera algo así:

``` javascript

{
  name: 'ValidationError',
  errors: {
    name: { message: 'Name is required' },
    email: { message: 'Email is invalid' }
  }
}

```

## ¿Qué hace la función?

## 1. Obtener todos los errores

``` javascript

Object.values(err.errors)

```

Convierte esto:

``` javascript

{
  name: {...},
  email: {...}
}

```

En esto:

``` javascript

[
  { message: 'Name is required' },
  { message: 'Email is invalid' }
]

```

## 2. Extraer los mensajes

``` javascript

.map(el => el.message)

```

Resultado:

``` javascript

[
  'Name is required',
  'Email is invalid'
]

```

## 3. Unirlos en un solo string

``` javascript

errors.join('. ')

```

Resultado:

```

Name is required. Email is invalid

```

## 4. Crear mensaje final

``` javascript

const message = `Invalid input data. ${errors.join('. ')}`;

```

Resultado:

```

Invalid input data. Name is required. Email is invalid

```

## 5. Convertir a AppError

``` javascript

return new AppError(message, 400);

```

## Resultado

``` json

{
  "status": "fail",
  "message": "Invalid input data. Name is required. Email is invalid"
}

```
