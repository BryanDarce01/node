# TCP/IP

El `TCP/IP` es el **conjunto de protocolos que permite que las computadoras se comuniquen entre sí a través de redes**, especialmente en Internet.

En otras palabras, **es el lenguaje que usan los dispositivos para enviarse datos.**

## Qué significa TCP/IP

`TCP/IP` no es un solo protocolo, sino dos principales:

- Transmission Control Protocol (TCP)

- Internet Protocol (IP)

Cada uno cumple un rol distinto.

**IP → Identifica y enruta**

El `IP` se encarga de:

- Asignar direcciones a los dispositivos (`IP addresses`).

- Enviar paquetes de datos al destino correcto.

Ejemplo de dirección IP:
```
192.168.1.1
```

Es como la dirección de una casa.

**TCP → Garantiza que los datos lleguen bien**

El TCP se encarga de:

- Dividir la información en paquetes

- Enviar los paquetes

- Verificar que lleguen completos y en orden

- Reenviar los que se pierdan

Es como un **servicio de mensajería certificado.**

# Cómo funciona TCP/IP

Supongamos que visitamos una página web.

- El navegador quiere entrar a un sitio.

- La computadora usa IP para encontrar el servidor.

- TCP divide los datos en paquetes pequeños.

- Los paquetes viajan por la red.

- El servidor responde enviando paquetes de vuelta.

- TCP los reordena para reconstruir la información.

Resultado:

Cargar la página web.

# Resumen

El `TCP/IP` es el conjunto de reglas que permite que los dispositivos se comuniquen en Internet.

- IP → encuentra el destino

- TCP → asegura que los datos lleguen completos
