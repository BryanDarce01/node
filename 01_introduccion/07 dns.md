# DNS

El **DNS (Domain Name System)** es el sistema que traduce nombres de dominio que los humanos entendemos en **direcciones IP que las computadoras** usan para comunicarse en Internet. 

# 1. La idea básica

Los humanos usamos nombres como:

```

Google → google.com

```

Pero las computadoras realmente se comunican con direcciones IP como:

```
142.250.190.78
```

## Algo interesante (muy usado en backend)

El **DNS** no solo traduce nombres, también se puede indicar:

```
servidores de correo

balanceo de carga

subdominios

CDN
```

Por ejemplo:

```
api.midominio.com

cdn.midominio.com

mail.midominio.com
```

Todo eso se configura en DNS.
