# ¿Qué es Node?

**Node.js is a JavaScript runtime built on Google’s open-source V8 JavaScript engine.**

# 1. Node.js

Es un **entorno de ejecución** (runtime) que permite ejecutar JavaScript fuera del navegador, por ejemplo:

- sen servidores

- sen scripts del sistema

- sen APIs

- sen aplicaciones backend

***Antes de Node.js, JavaScript prácticamente solo se ejecutaba en el navegador.***

# 2. JavaScript runtime

Un **runtime** es simplemente el **entorno** donde un lenguaje se ejecuta.

Ejemplos:

| Lenguaje                | Runtime         |
| ----------------------- | --------------- |
| JavaScript en navegador | Chrome, Firefox |
| JavaScript en servidor  | Node.js         |
| Java                    | JVM             |


***Node.js es entonces el entorno que ejecuta JavaScript en el servidor***

# 3. V8 JavaScript Engine

V8 es el motor de JavaScript creado por Google y usado en el navegador Google Chrome.

Su trabajo es:

- Leer código JavaScript

- Compilarlo

- Convertirlo en código máquina

- Ejecutarlo rápidamente

Es **extremadamente rápido**, por eso Node.js lo usa.

# Como funciona todo junto

Arquitectura simple

```

Código JavaScript
        ↓
Node.js Runtime
        ↓
V8 Engine
        ↓
Código máquina ejecutándose en el sistema

```

Node.js agrega además cosas que el navegador no tiene:

- acceso al filesystem

- red

- HTTP server

- procesos del sistema

``` javascript

    console.log("Hola Node.js");

```

Entonces ejecutamos

``` javascript

    node app.js

```

Y esto ocurre:

```

app.js
  ↓
Node.js
  ↓
V8 Engine
  ↓
CPU ejecuta el código

```
## La frase el inicio

***Node.js es un entorno que permite ejecutar JavaScript en el servidor usando el motor V8 de Google.***
