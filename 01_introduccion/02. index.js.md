# Módulo `http` 

``` javascript

    const http = require('http');

```

Este módulo permite crear servidores HTTP, es decir, servidores que pueden recibir peticiones desde un navegador o una API.

# Qué es `require`

`require()` es una función del sistema de módulos de Node llamada **CommonJS**.

Su función es:

- Cargar (importar) un módulo para poder usarlo en nuestro archivo.

En este caso estamos cargando `http`:

``` javascript

    require('http')

```

# Qué contiene el módulo `http`

El módulo `http` tiene varias funciones para trabajar con servidores web, por ejemplo:

- `createServer()`

- `request()`

- `get()`

Entonces la variable `http` pasa a contener todas esas funciones.

# Código completo

``` javascript
const http = require('http');

const server = http.createServer((req, res) => {
    res.end('Hola, mmundo!');
    url = req.url;
    console.log(`Direccion de la solicitud ${url}`); 
});

PUERTO = 3000

server.listen(PUERTO, () => {
    fecha = Date.now();
    console.log(`Servidor escuchando en el puerto ${PUERTO} desde ${fecha}`);
});

```
